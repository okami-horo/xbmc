# Quickstart: Android Danmaku Overlay

1. Checkout branch `001-dfm-android`.
2. Ensure Android SDK/NDK configured for Kodi build (see docs).
3. Build Android packaging (Gradle) and install APK on Android TV/emulator.
4. Place a same-basename danmaku file next to your video (e.g., movie.mp4 + movie.xml).
5. Start playback; enable Danmaku in settings if not auto-enabled. Verify smooth scrolling and sync across pause/seek/speed.
6. Adjust density/speed/font size/opacity/no-overlap and validate effects.
